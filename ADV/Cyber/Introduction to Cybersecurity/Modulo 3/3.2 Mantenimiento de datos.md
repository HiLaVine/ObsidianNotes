
### 3.2.1 Que es el cifrado

Es el proceso de conversión de la información a un formato que una parte no autorizada no puede leer. Solo una persona autorizada tiene la contraseña o clave de cifrado para poder acceder a estos datos.

### 3.2.2 Como se cifran los datos

Se utilizan programas de software para cifrar archivos, carpetas e incluso unidades externas.

El sistema de encriptación de archivos es una característica de Windows que permite encriptar datos. Esta vinculado a una cuenta especifica y solo el usuario que encripta puede acceder a estos datos.
