
Para buscar usando un tipo de expresión regular, podemos añadir un wildcard, en este caso el *, que se añadira dependiendo la búsqueda que estemos haciendo.

*field ==> cualquier cosa que tenga field después del wildcard  
field* ==> cualquier cosa que tenga field primero y después lo que sea.   
*field* ==> que el field este entre dos cosas cualquiera.

no tiene detección de mayúsculas.

Se pueden usar los boleanos AND OR y NOT para las búsquedas.

el AND es implícito, por lo que no se pone, los otros si es necesario colocarlos.

Y tiene un orden de evaluación. Primero el NOT, luego el OR y al final el AND y se puede usar paréntesis para controlar esto.
